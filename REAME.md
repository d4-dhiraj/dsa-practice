# DSA Practice 🚀

A personal repository to track my journey of learning and mastering **Data Structures and Algorithms (DSA)**. This repo contains solutions to problems from various platforms (LeetCode, Codeforces, GeeksforGeeks, HackerRank, etc.), organized by topic and difficulty.

## 📌 About

This repository serves as:
- A structured collection of DSA problems and their solutions
- A place to document approaches, time/space complexity, and learnings
- A progress tracker for consistent practice
- A reference for revising concepts before interviews

## 🗂️ Repository Structure

```
DSA-Practice/

├── Stacks/
├── Queues/
├── Trees/
│   ├── BinaryTree/
│   └── BST/

├── Greedy/
├── Backtracking/

├── Searching/
├── BitManipulation/
├── Math/
└── README.md
```

Each folder contains:
- Problem solutions named as `ProblemName.<extension>`
- A short explanation of the approach (in comments or a local notes file)
- Time & Space complexity analysis
